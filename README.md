# java-cli-maven-failsafe-spring-jacoco-cucumber-mockito-junit5-car-data

## Description
A POC for maven app using junit5
and cucumber and mockio framework
 with jacoco
and failsafe plugin.

## Tech stack
- java
- maven
	- spring
	- mockito
  - junit5
  - failsafe
  - jacoco
  - cucumber

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
